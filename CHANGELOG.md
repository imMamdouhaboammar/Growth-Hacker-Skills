# Changelog

All notable changes to Growth Hacker Skills are documented here.

## 2.0.0 - 2026-07-13

### Added

- Repository-level `growth-hacker-skills` router
- Neural Link Protocol for cross-skill routing
- Machine-readable `skill-graph.json`
- Shared `.growth-hacker/context.md` workflow contract
- Main Claude plugin skill under `skills/growth-hacker-skills/`
- Claude Code plugin manifest and marketplace catalog
- Claude plugin install, update, validate, and uninstall script
- SKILLS.sh repository configuration and install documentation
- GitHub Actions validation for skills, graph, distribution, and SKILLS.sh discovery
- Editorial Visual Engine for source-led visual content packs
- Permanent visual reference library
- Permanent corpus of 11 caption references
- ChatGPT Images prompt templates
- Google Flow silent motion templates
- Source Integrity Block and claim-separation rules
- Distribution, graph, and package validators

### Changed

- Rebranded the repository as Growth Hacker Skills
- Rebuilt the README around Mamdouh Aboammar and the connected architecture
- Replaced isolated workflow documentation with explicit node-to-node handoffs
- Updated the repository banner and visual identity
- Updated Claude marketplace metadata to the official schema
- Added migration metadata from the former `social-media-skills` plugin name
- Updated the MIT license notice to retain upstream attribution and identify Mamdouh's modifications
