# Creative Angle Library

Use this library to generate 3 to 6 source-specific options. Rename every angle so it describes the actual source.

| Angle family | Best for | Core question | Typical visual |
|---|---|---|---|
| Educational breakdown | Complex tools, repositories, frameworks | What must the reader understand first? | Field guide, layered explainer |
| Product or tool announcement | Launches and new releases | What changed and why does it matter? | Announcement poster, pipeline |
| Update or release | Version changes, new models, changelogs | What is new, available, limited, or time-sensitive? | Timeline, before and after |
| Promotional | Offers, free resources, waitlists | What value is available and what is the next step? | Benefit stack, CTA poster |
| Contrarian opinion | Strong thesis or market critique | Which common belief does the source challenge? | Split comparison, claim board |
| Step-by-step guide | Setup instructions and repeatable workflows | What should the reader do in order? | Numbered playbook |
| Comparison | Alternatives, versions, old versus new | What differs in capability, cost, process, or outcome? | Comparison board |
| Framework or system map | Connected components and dependencies | How do the pieces work together? | System map, architecture diagram |
| Lessons learned | Build logs, case notes, migrations | What changed after doing the work? | Before and after, lessons board |
| Use cases | Multi-purpose tools and models | What can the reader build or do? | Ranked use-case guide |
| Checklist | Preflight, QA, requirements | What must be true before execution? | Checklist board |
| Case study | Documented implementation and result | What happened, why, and what changed? | Problem-process-result board |
| Founder story | Personal build journey | What decision, tension, and lesson drive the story? | Editorial story map |
| Industry commentary | Trend or role shift | What pattern is forming and what does it imply? | Present versus future |

## Angle quality test

A usable angle passes all checks:

- Specific to the source
- Supported by at least three source elements, or one strong central source element
- Useful to a named audience
- Expressible in one visual format
- Different from the other proposed angles
- Does not depend on unsupported claims

## Bad versus useful

Bad:

- Educational
- Promotional
- Tool overview
- AI is changing work

Useful:

- The five workflows this repository gives a solo marketer
- What changed between the previous release and the current one
- The hidden operating model behind the tool's agent folders
- A practical setup map from installation to first output

## Recommendation scoring

Score each candidate from 1 to 5 on:

- Source support
- Reader value
- Visual clarity
- Novelty
- Mobile scannability

Recommend the highest total. Break ties using source support first, then visual clarity.
