# Caption Reference Corpus

This file stores the permanent caption examples supplied for the Editorial Visual Engine.

Use these examples as structural and pacing references only. Do not copy distinctive wording, personal claims, names, calls to action, dates, metrics, links, or product claims into a new caption unless the current source supports them.

The examples have been cleaned only to remove platform artefacts and to keep the repository English-only. Their structure, pacing, information density, and CTA patterns are intentionally preserved.

## Usage contract

When drafting a new caption:

1. Select one primary reference by structural fit.
2. Optionally borrow one secondary mechanism, such as a CTA shape or list rhythm.
3. Use facts only from the current source.
4. Follow `voice.md` over this corpus when a voice profile exists.
5. Never combine personal claims from a reference with the current user's identity.
6. Never preserve reference links, dates, product names, metrics, or access instructions unless they belong to the current source.
7. Do not imitate a reference author's distinctive wording.
8. Record the chosen reference ID in the internal creative direction, not in the published caption.
