# Example: Tool Launch to Announcement Pipeline

## User request

> Create a viral editorial launch graphic from this AI tool page. Include motion and an X thread.

## Expected route

1. Read the product page, documentation, pricing or access notes, examples, and release information inside the source.
2. Identify the launch promise, workflow stages, intended users, limits, and access condition.
3. Recommend EV-03 for a multi-stage launch or EV-07 for one integration and one CTA.
4. Lock exact product names, dates, prices, and limits.
5. Generate a motion poster prompt with stable text and connector animation.
6. Use an X thread only when several workflow stages require explanation.

## Failure conditions

- Calling the tool free without a supported access statement
- Claiming performance outcomes not shown in the source
- Adding channel logos that do not appear in the source
- Letting Google Flow regenerate the copy
